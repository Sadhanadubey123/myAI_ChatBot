# How to Run

### 1. Start the mock MDSO API
How to Run
1- Start mock API:
python mdso_mock/server.py
2- Start backend application:
python app.py
3- Start UI:
streamlit run ui/streamlit_ui.py



# Send a request
curl -X POST http://127.0.0.1:8000/chat \
     -H "Content-Type: application/json" \
     -d '{"message":"provision a new service"}'

How to Run the Interactive UI
1. Make sure both servers are running:
- Mock MDSO API → python mdso_mock/server.py
- Chatbot backend → python app.py

2. Start the Streamlit UI:
streamlit run ui/streamlit_ui.py

3. Open the browser link (usually http://localhost:8501) 
and you’ll see a chat window where you can type messages and get responses.
